# 功能概览

首页核心功能包括分组导航、添加网址、卡片图标自动获取、标签搜索、最近使用、星标置顶、链接健康检查、主题切换、背景与粒子特效。添加网址时支持直接新建分组，卡片和分组都可以设置背景媒体。无内容的首页卡片会隐藏，避免空状态影响页面观感。

日历支持月视图、周视图、列表、统计、任务重复规则、提醒和 ICS 导入导出。首页的即将到来卡片会跟随设置显示，日历图标和天气联动会随主题切换保持统一视觉。页面缩小时，日历网格使用更稳定的尺寸约束，避免日期堆叠。

AI 助手支持多供应商、模型拉取、人设、附件、语音输入、Markdown、图片返回预览与保存。AI 输出导航指令时，会生成操作卡；当前版本新增变更预览和应用前快照，用户可以在应用后撤销本次 AI 操作。

设置页包含账号、安全、外观、背景、特效、组件、天气、数据管理、同步迁移、扩展模块。数据管理中可以查看 SQLite key、媒体文件、快照、ZIP 导入导出和孤儿媒体清理。命令面板可用 Ctrl/Cmd+Shift+P 打开，用于快速执行常用动作。
